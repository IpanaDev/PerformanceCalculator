# Q/A
* How am I supposed to be up to date?
1. Go to SBRW Launcher. 
2. Update your game.
3. Delete "cars.txt". 
4. Reopen tool and it will parse latest cars and performances.

* Can I make my own tune?
1. Choose "Custom Parts" in selection box.
